# Farbduell

A color based card game – free, ad-free and configurable. Installable as a PWA, runs fully offline against 1–3 bots.

[Deutsche Version](README-DE.md)

## Settings

- 1–3 bots, light (default) / dark theme
- **Stack +2**: counter an incoming +2 with your own, penalty pile keeps growing
- **Draw until you match**: draw cards until a playable one comes up
- **Turn time limit**: off (default) / 30 / 60 / 90 s – time runs out, a card is drawn automatically
- 2-player rule: reverse acts like skip
