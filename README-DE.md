# Farbduell

Das farbbasierte Kartenspiel – kostenlos, werbefrei und konfigurierbar. Als PWA installierbar, läuft komplett offline gegen 1–3 Bots.

## Einstellungen

- 1–3 Bots, helles (Standard) / dunkles Theme
- **+2 stapeln**: Konter mit eigener +2, Strafstapel wandert weiter
- **Ziehen, bis es passt**
- **Zeitlimit pro Zug**: aus (Standard) / 30 / 60 / 90 s – läuft die Zeit ab, wird automatisch gezogen
- 2-Spieler-Sonderregel: Richtungswechsel wirkt wie Aussetzen
